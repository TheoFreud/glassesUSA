Installation

Learn how to include the styling in your project.

To install the styling and scripts, include the following code in the 
<head>‹/head>
section of your project 
                
                    <!-- CSS Styling -->

                    <link rel="stylesheet"
                    href="https://cdn.jsdelivr.net/gh/TheoFreud/glassesUSA@latest/css/index.css">
                    

                    <!-- Scripts - Required if you are using any form elements/buttons -->

                    <script
                    src="https://cdn.jsdelivr.net/gh/TheoFreud/glassesUSA@latest/scripts/index.js"></script>
